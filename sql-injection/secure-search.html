🔒 How Prepared Statements Prevent SQL Injection

1. Vulnerable Code (DON'T USE):

// VULNERABLE - Direct string concatenation
$sql = "SELECT * FROM users WHERE username = '" . $username . "' AND password = '" . $password . "';";
$result = mysqli_query($connection, $sql);

2. Secure Code with Prepared Statements:

// SECURE - Prepared statement with parameter binding
$sql = "SELECT * FROM users WHERE username = ? AND password = ?";
$stmt = $connection->prepare($sql);
$stmt->bind_param("ss", $username, $password);
$stmt->execute();
$result = $stmt->get_result();

3. Why Prepared Statements are Secure:

  • SQL Structure is Pre-compiled: The database compiles the SQL statement before user input is added
  • Parameters are Bound Safely: User input is treated as data only, never as SQL code
  • Special Characters are Escaped: SQL metacharacters lose their special meaning
  • Type Safety: Parameters are bound with specific data types (string, integer, etc.)

4. Injection Attempts That Will Fail:

Test 1 - Comment Injection: Username: admin' -- Password: anything Result: Searches for literal username "admin' --" (fails safely)
Test 2 - OR Condition: Username: ' OR '1'='1 Password: anything Result: Searches for literal username "' OR '1'='1" (fails safely)
Test 3 - UNION Attack: Username: ' UNION SELECT * FROM users -- Password: anything Result: Searches for literal username "' UNION SELECT * FROM users --" (fails safely)

5. Complete PHP Implementation:

<?php
// secure_login.php - Complete implementation

$servername = "localhost";
$db_username = "root";
$db_password = "";
$dbname = "clinicDB";

// Create connection
$conn = new mysqli($servername, $db_username, $db_password, $dbname);

if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}

if ($_POST) {
    $username = $_POST['username'];
    $password = $_POST['password'];

    // SECURE: Prepared statement
    $sql = "SELECT id, username FROM users WHERE username = ? AND password = ?";
    $stmt = $conn->prepare($sql);

    if ($stmt) {
        // Bind parameters (s = string)
        $stmt->bind_param("ss", $username, $password);
        $stmt->execute();
        $result = $stmt->get_result();

        if ($result->num_rows > 0) {
            $user = $result->fetch_assoc();
            echo "Login successful! Welcome " . htmlspecialchars($user['username']);
        } else {
            echo "Invalid credentials.";
        }
        $stmt->close();
    }
}
$conn->close();
?>
